    ⚙ Synk via GitHub

    Appen läser och sparar paletten.json i ett privat GitHub-repo, så att dina enheter delar samma projekt och färger. Token sparas bara lokalt på den här enheten — aldrig i koden eller i synkfilen.
    Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write. Flera appar kan dela samma repo — varje app sparar sin egen fil.
