    ⚙ Synk via GitHub

    Banden sparar dina personer och uppskattningar i en privat GitHub-fil (banden.json), så att telefon och dator visar samma sak. Din token sparas bara på den här enheten — aldrig i koden.
    Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST det repot, med Contents: Read and write.
