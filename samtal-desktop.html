⚙ Teamsynk via GitHub

Appen läser och sparar data.json i ett privat GitHub-repo så att hela teamet ser samma medarbetare och samtal. Token sparas bara lokalt på den här enheten — aldrig i koden. Samma repo som mobilappen använder.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
