✨ AI-inställningar

AI-förslagen görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje förslag kostar några ören från ditt API-saldo.

Utan nyckel fungerar allt annat i appen som vanligt — bara AI-förslagen behöver den.

⟲ Synk mellan enheter

Familjen sparar familjen.json i ett privat GitHub-repo, så att samma familjekarta, kalender och minnen finns på alla dina enheter — telefon och dator. Använd samma användarnamn och repo här som i mobilappen. Token sparas bara lokalt på den här enheten.

github.com → Settings → Developer settings → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
