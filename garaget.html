⚙ Synk via GitHub

Appen läser och sparar garaget.json i ett privat GitHub-repo, så att din mobil och dator alltid visar samma bil. Din token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till enbart data-repot, med behörigheten Contents: Read and write.
