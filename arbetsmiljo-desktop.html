Teamsynk via GitHub

Risker och åtgärder delas med teamet via en data-fil i ett privat GitHub-repo. Dina personliga checkar, batteri, NPF-loggar och AI-chatt delas aldrig — de stannar i din enhet. Token sparas bara lokalt.
Skapas på github.com → Settings → Developer settings → Fine-grained tokens. Ge åtkomst till endast detta repo, med Contents: Read and write.
