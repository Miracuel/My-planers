⚙ Synk via GitHub

Appen läser och sparar hemkomst.json i ett privat GitHub-repo så att dina enheter delar samma rutin och historik. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
