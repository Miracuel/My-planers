⚙ Synk via GitHub

Appen läser och sparar medicin.json i ett privat GitHub-repo, så att samma mediciner och doser syns på alla dina enheter. Du kan dela samma repo med dina andra appar — varje app har sin egen fil. Token sparas bara lokalt på den här enheten — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.

✨ AI-inställningar

Välj leverantör och klistra in din egen API-nyckel. Nyckeln sparas bara lokalt på den här enheten och skickas aldrig till synken.

Skapas på console.anthropic.com → API Keys.
