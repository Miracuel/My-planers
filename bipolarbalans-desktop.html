✨ AI-stöd

AI-funktionerna körs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt på den här enheten och följer inte med i säkerhetskopior. Varje anrop kostar några ören av ditt API-saldo.

Inställningar

Lokal
BipolärBalans är ett stöd för självinsikt, struktur och kommunikation med vård och närstående. Det ersätter inte vård, diagnos eller medicinsk behandling. Vid akut fara: ring 112. Stöd dygnet runt: Självmordslinjen 90101.

⚙ Synk via GitHub

Appen läser och sparar bipolarbalans.json i ett privat GitHub-repo (samma repo kan delas med dina andra appar). Token sparas bara lokalt på den här enheten.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
