⚙️ Inställningar

Datorn läser och sparar it-koordinator.json i samma privata GitHub-repo som mobilappen — så telefon och dator delar ärenden, personal, utrustning och kunskapsbank. Token sparas bara lokalt på den här enheten.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Åtkomst till ENDAST data-repot, Contents: Read and write. Använd samma repo som i mobilen.

AI-funktionerna körs av Claude via din egen API-nyckel från console.anthropic.com. Sparas bara lokalt, synkas inte.
