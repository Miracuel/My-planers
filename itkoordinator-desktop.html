⚙️ Inställningar

Appen läser och sparar it-koordinator.json i ett privat GitHub-repo, så att din telefon och datorversionen visar samma ärenden, personal, utrustning och kunskapsbank. Token sparas bara lokalt på den här enheten.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.

AI-funktionerna körs av Claude via din egen API-nyckel. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt och följer inte med i säkerhetskopior eller synk.
