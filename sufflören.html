Synk mellan enheter Lokal

Sparar sufflören.json i ett privat GitHub-repo så att mobil och dator ser samma kunskapsbas. Token sparas bara lokalt på den här enheten — aldrig i koden eller i filen.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge Contents: Read and write på just det repot.

AI-assistent

Svaren görs av Claude via din egen API-nyckel från Anthropic (console.anthropic.com → API keys). Nyckeln sparas bara lokalt på enheten och synkas aldrig.
