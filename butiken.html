⚙ Synk via GitHub

Appen läser och sparar butiken.json i ett privat GitHub-repo så att telefon och dator delar samma data. Token sparas bara lokalt på den här enheten — aldrig i koden.
github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
