GitHub-synk Lokal
Synka mat, sömn, blöjor, tillväxt och milstolpar mellan era enheter via ett privat GitHub-repo. Två vårdnadshavare ser samma data — det som loggas på en enhet dyker upp på den andra.
Inställningar
Allt sparas bara här på din enhet. Inget skickas någonstans.

⚙ Synk via GitHub

Appen läser och sparar babykoll-data.json i ett privat GitHub-repo så att hela familjen ser samma data. Token sparas bara lokalt på den här enheten — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge behörigheten Contents: Read and write på data-repot.
