✨ AI-nyckel

AI-funktionerna använder din egen Anthropic-nyckel. Den sparas bara lokalt på den här enheten — aldrig i koden eller i synken.

Skapas på console.anthropic.com → API Keys.

⚙ Synk via GitHub

Appen läser och sparar ledare.json i ett privat GitHub-repo, så att samma data finns på alla dina enheter (flera appar kan dela samma repo). Token sparas bara lokalt — aldrig i koden.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, Contents: Read and write.
