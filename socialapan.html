⚙ Synk via GitHub

Social Apan läser och sparar socialapan.json i ett privat GitHub-repo, så att din telefon och dator visar samma data. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
