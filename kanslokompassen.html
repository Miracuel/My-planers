⚙ Synk via GitHub

Appen läser och sparar data.json i ett privat GitHub-repo, så att dina känslor följer med mellan dina enheter. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
