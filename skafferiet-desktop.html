⚙ Synk via GitHub

Skafferiet sparar din data i en privat fil (skafferiet.json) i ett GitHub-repo, så att telefonen och datorn ser samma skafferi. Token sparas bara här på enheten.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till endast repot, med Contents: Read and write.
