Kom igång

Det svåra är inte uppgiften — det är att börja. Du behöver inte göra allt. Bara första rörelsen.
1. Välj det minsta möjliga första steget. Inte hela uppgiften — bara att öppna dokumentet, eller hämta pappret.
2. Kroppen först: ställ dig upp, drick lite vatten, sätt på lurarna.
3. Säg högt vad du ska göra. Det räknas att börja — inte att bli klar.
4. Sätt timern på 2 minuter. När den ringer får du sluta. Oftast vill du inte.

Steg

Mindre steg = lättare att börja. Ett ord räcker.

Välj något att göra nu

En sak i taget. Resten väntar snällt i listan.

⚙ Synk via GitHub

Appen läser och sparar igang-data.json i ett privat GitHub-repo, så telefon och dator delar samma data. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
