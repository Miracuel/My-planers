⚙ Synk via GitHub

Insikten sparar sin data i insikten-data.json i ett privat GitHub-repo, så att du ser samma analyser på alla enheter. Token sparas bara lokalt — aldrig i koden.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST repot, med Contents: Read and write.
