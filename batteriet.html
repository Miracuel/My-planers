⚙ Synk via GitHub

Batteriet läser och sparar batteriet.json i ett privat GitHub-repo, så att mobilen och datorn ser samma data. Token sparas bara lokalt på den här enheten — aldrig i koden. Tänk på att mående och symtom är känsligt: använd ett repo som bara du kommer åt.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
