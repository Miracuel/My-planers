⚙ Synk via GitHub

Appen läser och sparar skuldfri-data.json i ett privat GitHub-repo, så att samma skulder och betalningar finns på alla dina enheter. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till endast data-repot, med Contents: Read and write.
