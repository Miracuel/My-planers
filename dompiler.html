⚙ Synk via GitHub

Appen läser och sparar dompiler.json i ett privat GitHub-repo, så att telefon och dator visar samma dompiler. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till endast data-repot, med Contents: Read and write.
