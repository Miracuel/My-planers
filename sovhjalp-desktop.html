☁ Synk mellan enheter

Appen sparar din data som sovhjalp.json i ett privat GitHub-repo, så att telefon och dator visar samma sak. Token sparas bara här på enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Fine-grained tokens. Ge den Contents: Read and write på repot.
🔒 End-to-end-krypteringKrypterar allt med ett lösenord innan det lämnar enheten. Rekommenderas för känsliga uppgifter.
Måste vara exakt samma på alla enheter. Tappar du det går datan inte att läsa — det finns ingen återställning.
Flera appar kan dela samma privata repo — varje app sparar sin egen fil. Senaste ändringen vinner om du redigerar på två enheter samtidigt.
