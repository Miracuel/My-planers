✨ AI-uppdrag

Uppdragen genereras av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje generering kostar några ören från ditt API-saldo.

✨ Nyheter & funktioner

⟳ Synk via GitHub

Appen sparar din speldata i filen data.json i ett privat GitHub-repo, så att telefon och dator delar samma äventyr. Din token sparas bara lokalt på den här enheten — aldrig i koden eller i säkerhetskopiorna.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens, med åtkomst till just det repot.
