✨ AI-uppdrag

Uppdragen genereras av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje generering kostar några ören från ditt API-saldo.

\u27f3 Synk via GitHub

Appen sparar din speldata i filen data.json i ett privat GitHub-repo, s\u00e5 att telefon och dator delar samma \u00e4ventyr. Din token sparas bara lokalt p\u00e5 den h\u00e4r enheten \u2014 aldrig i koden eller i s\u00e4kerhetskopiorna.

Skapas p\u00e5 github.com \u2192 Settings \u2192 Developer settings \u2192 Personal access tokens \u2192 Fine-grained tokens, med \u00e5tkomst till just det repot.
