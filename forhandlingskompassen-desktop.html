⚙️ Synk via GitHub

Appen läser och sparar forhandlingskompassen.json i ett privat GitHub-repo så att teamet ser samma data. Token sparas bara lokalt på enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till endast data-repot, med Contents: Read and write.

✨ AI-nyckel

Klistra in din Anthropic API-nyckel. Den sparas bara lokalt på den här enheten och används för AI-stödet (frågeförslag, riskbedömning, protokollutkast, likabehandlingsgranskning).
