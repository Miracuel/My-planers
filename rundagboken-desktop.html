⚙ Synk via GitHub

Appen läser och sparar rundagboken.json i ett privat GitHub-repo så att din runresa följer med mellan enheter. Token sparas bara lokalt på den här enheten — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot med Contents: Read and write.

☀ AI-nyckel

AI-tolkningar görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt och följer inte med i säkerhetskopior. Varje tolkning kostar några ören.
