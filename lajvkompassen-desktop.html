✨ AI-nyckel

AI-hjälpen drivs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje fråga kostar några ören från ditt API-saldo.

⚙ Synk via GitHub

Appen läser och sparar data.json i ett privat GitHub-repo så att din data följer med mellan enheter. Token sparas bara lokalt — aldrig i koden.

Settings → Developer settings → Fine-grained tokens. Ge åtkomst till ENDAST data-repot med Contents: Read and write.
