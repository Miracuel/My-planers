✨ AI-meddelanden

AI-texterna skrivs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din telefon och följer inte med i säkerhetskopior. Varje text kostar några ören från ditt API-saldo. Appen funkar fint utan nyckel — då används färdiga förslag istället.

☁︎ Synk mellan enheter

Vännerna läser och sparar vannerna.json i ett privat GitHub-repo, så att din telefon och din dator visar samma vänner. Token sparas bara lokalt på den här enheten — aldrig i koden och inte i säkerhetskopior.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till endast data-repot.
