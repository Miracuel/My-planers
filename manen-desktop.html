☁ Synk via GitHub

Månen läser och sparar manen.json i ett privat GitHub-repo, så att anteckningar och journal följer med mellan dina enheter. Samma data-repo som Appkollen och de andra apparna funkar fint — filerna krockar inte. Token sparas bara lokalt på den här enheten, aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge den läs- och skrivrätt på Contents för data-repot. Den enhet som sparar sist vinner.
