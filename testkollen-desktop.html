⚙ Synk via GitHub

Testkollen läser och sparar testkollen.json i ett privat repo så att flera testare ser samma protokoll. Token sparas bara här på enheten — aldrig i koden eller i synk.
När du kopplar hittar och läser Testkollen alla changelog-filer i repot automatiskt (t.ex. appkollen-changelog.json, centralen-changelog.json) och pingar när nya funktioner dykt upp. Den lägger samtidigt till sin egen testkollen-changelog.json i repot, så Centralen och Appkollen kan läsa den utan manuell uppladdning.

✨ AI-testfall

Förslagen genereras av Claude via din egen API-nyckel från Anthropic (console.anthropic.com → API keys). Nyckeln sparas bara lokalt och följer inte med i synk eller säkerhetskopior. Varje förslag kostar några ören.
