✨ API-nyckel

Coachen drivs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer aldrig med i säkerhetskopior eller synk. Varje fråga kostar några ören av ditt API-saldo.

⚙ Synk via GitHub

Appen läser och sparar lovergirl.json i ett privat GitHub-repo, så att dina enheter ser samma sak. Token sparas bara lokalt på den här enheten — aldrig i synken. Andra appar i sviten kan dela samma repo (egen filnamn per app).

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
