⚙ Synk via GitHub

Appen sparar en privat data.json i ett GitHub-repo så att telefon och dator visar samma logg. Din token sparas bara i den här enheten — aldrig i koden. Inget loss av loggade saker: vid synk slås loggarna ihop.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
