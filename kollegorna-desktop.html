🤝 Kollegorna
Lokal
Välj en kollega till vänster, eller lägg till en ny.
Få hjälp att formulera ett meddelande som passar personens sätt att kommunicera. Mindre grubbel — tydligare ord.

🔗 Synk mellan enheter

Appen sparar kollegorna.json i ett privat GitHub-repo, så samma data finns på telefon och dator. Ange samma repo som i mobilappen. Token sparas bara lokalt på den här enheten.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Endast data-repot, Contents: Read and write.

✨ AI-funktioner

Sammanfattningar och formuleringsförslag görs av Claude via din egen API-nyckel från Anthropic (console.anthropic.com → API keys). Nyckeln sparas bara lokalt på den här enheten.
