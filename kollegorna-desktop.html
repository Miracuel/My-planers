✨ AI-funktioner

Sammanfattningar och formuleringsförslag görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje anrop kostar några ören från ditt API-saldo.

🔗 Synk mellan enheter

Appen sparar kollegorna.json i ett privat GitHub-repo, så att samma data finns på telefon och dator. Token sparas bara lokalt på den här enheten — aldrig i koden eller i säkerhetskopior.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
