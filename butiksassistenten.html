⚙ Synk via GitHub

Appen läser och sparar butiksassistenten.json i ett privat GitHub-repo, så telefon och desktop ser samma data. Token sparas bara lokalt på enheten.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
