⚙ Synk via GitHub

Djurkollen läser och sparar djurkollen.json i ett privat GitHub-repo, så mobil och dator ser samma data. Token sparas bara lokalt på den här enheten — aldrig i koden eller i synkfilen.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
