⚙ Synk via GitHub

Appen läser och sparar ledstangen.json i ett GitHub-repo så att dina enheter delar samma data. Din token sparas bara här på enheten — aldrig i koden.
⚠ Repot måste vara privat — Team-fliken innehåller känsliga personuppgifter.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens, med åtkomst till repot och behörighet Contents: Read and write.
