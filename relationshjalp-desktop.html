☁ Synk via GitHub

Appen läser och sparar relationshjalp-data.json i ett privat GitHub-repo, så att ni båda ser samma data på alla enheter. Token sparas bara lokalt på den här enheten — aldrig i koden eller i säkerhetskopian.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge den Contents: Read and write på just det repot. Använd samma uppgifter på din partners enhet och på datorn.

✨ AI-funktioner

AI-svaren görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje svar kostar några ören från ditt API-saldo.
