✨ API-nyckel

AI-prioriteringen görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din webbläsare och följer inte med i säkerhetskopior. Varje förslag kostar några ören från ditt API-saldo. Utan nyckel kan du alltid använda lokal prioritering.

⚙ Synk via GitHub

Bygget läser och sparar bygget.json i ett privat GitHub-repo, så att telefon och dator delar samma projekt. Token sparas bara lokalt på den här enheten — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
