⚙ Synk mellan enheter

Idécentralen läser och sparar idecentralen.json i ett privat GitHub-repo, så att dator och telefon delar samma idéer. Din token sparas bara här på enheten — aldrig i koden eller hos någon annan.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
