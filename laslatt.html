⟳ Synk via GitHub

Appen sparar dina anteckningar och ord i en privat fil i ett GitHub-repo, så att de följer med mellan dina enheter. Din token sparas bara lokalt – aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
