⚙ Synk via GitHub

Appen sparar data.json i ett privat GitHub-repo så att samma data nås på flera enheter. Token sparas bara lokalt — aldrig i koden eller backupen.

github.com → Settings → Developer settings → Fine-grained tokens. Ge åtkomst till ENDAST detta repo med Contents: Read and write.

🤖 Anthropic API-nyckel

Tolkningarna görs av Claude via din egen nyckel från console.anthropic.com → API keys. Nyckeln sparas bara lokalt och följer inte med i backupen. Varje fråga kostar några ören av ditt API-saldo.
