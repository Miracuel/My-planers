⚙ Synk via GitHub

Appen läser och sparar kattkompis-data.json i ett privat GitHub-repo så att dina enheter delar samma data. Token sparas bara lokalt på den här enheten — aldrig i koden. Foton synkas inte, de stannar lokalt.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST detta repo med Contents: Read and write.
