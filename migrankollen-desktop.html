⚙ Synk via GitHub

Appen läser och sparar migran-data.json i ett privat GitHub-repo så att samma data finns på alla dina enheter. Token sparas bara lokalt — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
