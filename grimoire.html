⚙ Synk via GitHub

Skuggdagboken läses och sparas som grimoire.json i ett privat GitHub-repo — så ser mobilen och datorn samma sidor. Token sparas bara lokalt på den här enheten, aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write. Samma repo som Jobbplaneringen går bra — filerna heter olika saker.

🎨 Tema

✨ AI-hjälp

Ritualer och tolkningar skapas med din egen API-nyckel. Välj tjänst nedan. Nycklarna sparas bara lokalt på enheten, följer inte med i synk eller säkerhetskopior, och varje anrop kostar några ören från ditt saldo.
