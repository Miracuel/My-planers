☁ Synk mellan enheter

Rörelse kan spara din data i ett privat GitHub-repo, så att mobilen och datorn alltid visar samma pass, planering och gym. Din token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till endast data-repot, med Contents: Read and write.
