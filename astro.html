⚙ Synk via GitHub

Appen läser och sparar astro.json i ett privat GitHub-repo, så att t.ex. mobilen och datorn ser samma kartor. Token sparas bara lokalt på den här enheten — aldrig i koden. Födelsedata är personligt: använd ett repo som bara du kommer åt.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
