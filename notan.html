⚙ Synk via GitHub

Appen läser och sparar data.json i ett privat GitHub-repo så att dina enheter ser samma siffror. Token sparas bara lokalt på den här enheten.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
