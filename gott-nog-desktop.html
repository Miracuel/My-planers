⚙ Synk via GitHub

Appen sparar din data som gottnog.json i ett privat GitHub-repo, så att telefon och dator visar samma sak. Din token sparas bara lokalt på den här enheten — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
