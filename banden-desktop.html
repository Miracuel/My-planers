⚙ Synk via GitHub

Banden sparar dina personer och uppskattningar i en privat GitHub-fil (banden.json), så att dator och telefon visar samma sak. Token sparas bara på den här enheten — aldrig i koden.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST det repot, med Contents: Read and write. Använd samma repo i mobilappen.
