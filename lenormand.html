✨ AI-inställningar

Tolkningarna görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt på den här enheten och följer inte med i säkerhetskopior.

💾 Synk via GitHub

Appen läser och sparar lenormand-data.json i ett privat GitHub-repo så att dina enheter delar samma data. Token sparas bara lokalt — aldrig i koden.

github.com → Settings → Developer settings → Personal access tokens → Fine-grained. Ge åtkomst till ENDAST data-repot, Contents: Read and write.
