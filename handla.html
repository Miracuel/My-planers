⚙ Synk via GitHub

Appen läser och sparar handla.json i ett privat GitHub-repo, så att telefon och dator delar samma lista. Din token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Fine-grained tokens. Ge åtkomst till enbart detta repo med Contents: Read and write.
