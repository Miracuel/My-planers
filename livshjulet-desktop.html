⚙ Synk via GitHub

Appen läser och sparar livshjulet-data.json i ett privat GitHub-repo, så att telefon och dator delar samma hjul. Token sparas bara lokalt på den här enheten. Senast sparade ändringen vinner.

✨ AI-coach

Coachningen görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt och följer inte med i säkerhetskopior.
