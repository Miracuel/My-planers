⚙ Synk via GitHub

Appen läser och sparar livshjulet.json i ett privat GitHub-repo, så att telefon och dator delar samma hjul. Token sparas bara lokalt på den här enheten — aldrig i koden eller i säkerhetskopior. Drömlistor slås ihop från alla enheter; övriga fält följer den senaste ändringen.

✨ AI-coach

Coachningen görs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje svar kostar några ören från ditt API-saldo.
