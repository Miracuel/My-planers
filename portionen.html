⚙ Synk via GitHub

Portionen kan spara en privat portionen.json i ett GitHub-repo, så att telefon och dator visar samma data. Din token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST detta repo, med Contents: Read and write.
