Viktigt att veta

Var saker finns och vad som behöver göras. Skriv aldrig lösenord i klartext här – hänvisa hellre till var de finns (t.ex. lösenordshanteraren).

Kontakter

Personer som bör få veta, och som kan hjälpa till. Tryck på telefonsymbolen för att ringa.

Inställningar

Synk (privat GitHub)
Synkar krypterat mellan dina enheter via ett privat GitHub-repo. Lägg in ägare, repo och en personlig token (lagras bara här på enheten).
AI-hjälp för breven
Frivilligt. Hjälper dig att börja formulera ett brev. Välj tjänst och lägg in din egen nyckel – den lagras bara här på enheten.
Utseende
Data
Allt lagras på din enhet. Tänk på att berätta för någon du litar på var den här informationen finns – annars hjälper den ingen.
