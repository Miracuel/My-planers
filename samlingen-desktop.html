⚙ Synk via GitHub

Samlingen läser och sparar samlingen.json i ett privat GitHub-repo, så att telefon och dator delar samma samling. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
