⚙ Synk via GitHub

Bygget läser och sparar bygget.json i ett privat GitHub-repo, så att telefon och dator delar samma projekt. Token sparas bara lokalt på den här enheten — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write. Använd samma repo som i mobilappen.

✨ API-nyckel

AI-prioriteringen görs av Claude via din egen API-nyckel från Anthropic (console.anthropic.com → API keys). Nyckeln sparas bara lokalt och följer inte med i säkerhetskopior. Utan nyckel kan du alltid prioritera lokalt.
