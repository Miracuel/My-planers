⚙ Synk via GitHub

Appen läser och sparar projekt.json i ett privat GitHub-repo, så att samma data finns på alla dina enheter. Token sparas bara lokalt på den här enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Fine-grained tokens. Ge åtkomst till ENDAST data-repot med Contents: Read and write.
