    ✨ AI-nyckel

    Råden ges av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt på den här enheten och följer aldrig med i säkerhetskopior eller synk.

    ⚙ Synk via GitHub

    Appen läser och sparar data.json i ett privat GitHub-repo, så att din dator och mobil (och hela familjen) delar samma trädgårdsplan. Token sparas bara lokalt — aldrig i koden eller säkerhetskopian.

    Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.

    All data sparas lokalt på din enhet och i ditt eget privata GitHub-repo. Inget skickas någon annanstans.
