Inställningar

Appen läser och sparar appkollen.json i ett privat GitHub-repo — samma repo som jobbplaneringen funkar fint, filerna krockar inte. Token sparas bara lokalt på den här enheten.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
