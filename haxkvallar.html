✨ AI-inställningar

AI-hjälpen drivs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt i din enhet och följer inte med i säkerhetskopior. Varje förfrågan kostar några ören.

☁ GitHub-synk

Synka mellan dina enheter (och med cirkeln) via ett privat GitHub-repo. Skapa en fine-grained token med Contents: Read and write för repot. Allt sparas annars bara lokalt.
