✨ AI-nyckel

Coachen och träningen drivs av Claude via din egen API-nyckel från Anthropic. Skapa en på console.anthropic.com → API keys. Nyckeln sparas bara lokalt på den här enheten och följer inte med i säkerhetskopior eller synk.

⚙ Synk via GitHub

Appen läser och sparar data.json i ett privat GitHub-repo så att dina förhandlingar finns på alla enheter. Token sparas bara lokalt — aldrig i koden.

Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST data-repot, med Contents: Read and write.
