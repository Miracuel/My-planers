Checklista årsbokslut
BAS-baserad kontoplan. Lägg till egna konton vid behov.
Standarddatum för småföretag (deklaration & betalning ofta den 12:e). Justera vid behov och kontrollera alltid mot Skatteverket.
Direktkoppling till bank, Fortnox/Visma, Skatteverket, Swish och Stripe/Klarna kräver en server med OAuth och kan inte göras säkert i en fristående HTML-fil. Det praktiska alternativet här är att importera kontoutdrag som CSV.
AI-funktionerna (tolka kvitto, föreslå konto, förklara, hitta avvikelser) använder din egen Anthropic API-nyckel. Den sparas bara lokalt på den här enheten och skickas direkt till Claude.
Appen läser och sparar kassaboken.json i ett privat GitHub-repo så att telefon och dator delar samma bokföring. Token sparas bara lokalt på den här enheten — aldrig i koden. Använd ett eget repo för Kassaboken (inte samma som andra appar).
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till ENDAST det repot.
Exporterar all data som JSON. Ta en kopia regelbundet.
Kassaboken hjälper dig sköta din egen bokföring men ersätter inte en auktoriserad redovisningskonsult. Du ansvarar för att uppgifterna stämmer.
