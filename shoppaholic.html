⟳ Synk mellan enheter

Appen sparar din data som shoppaholic.json i ett privat GitHub-repo, så att telefon och dator alltid visar samma listor. Din token sparas bara här på enheten — aldrig i koden.
Skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till enbart data-repot.
