⟳ Synk via GitHub

Måsten läser och sparar masten.json i ett privat GitHub-repo, så samma måsten finns på alla dina enheter (dator och telefon). Token sparas bara här på enheten — aldrig i koden.
Token skapas på github.com → Settings → Developer settings → Personal access tokens → Fine-grained tokens. Ge åtkomst till endast data-repot, med Contents: Read and write.
